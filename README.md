# YOLO Mobile In-Use Detection

Real-time detection of "phone in use" (hand holding/using mobile phone) with YOLOv11.

## Features
- Trained from scratch on custom dataset (~500 images)
- YOLO11m backbone
- GPU accelerated training & inference
- ONNX export support
- Webcam / IP camera / video file support
